## 第一优先级：高级数据分析

### 1. 力量进步曲线

**目标**：展示用户各动作的力量进步趋势

**技术方案**：
- [ ] 设计力量记录表结构（exercise_progress）
- [ ] 创建力量数据录入 API
- [ ] 力量进步折线图（Recharts）
- [ ] Dashboard 新增力量分析卡片

### 2. 身体围度记录扩展

**目标**：完善围度测量数据展示

**技术方案**：
- [ ] BodyMetricsCard 显示所有围度数据（已有字段：chest/waist/hip/arm/thigh）
- [ ] 围度变化趋势图
- [ ] 围度数据表格视图

### 3. 前后对比照片

**目标**：用户可上传对比照片

**技术方案**：
- [ ] 设计照片存储方案（Cloudflare R2 或外部存储）
- [ ] 照片上传 API
- [ ] 对比照片展示组件
- [ ] 时间线照片浏览
