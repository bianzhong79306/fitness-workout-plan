# FitPlan Pro 2.0 - 需求文档（推倒重来版）

> 创建时间：2026-04-11
> 产品定位：智能健身知识平台
> 核心理念：教你"怎么练"，而不是"陪你练"

---

## 一、产品定位与核心理念

### 问题诊断

| 问题 | 旧版本现状 | 核心缺陷 |
|------|------------|----------|
| 训练完成 | 点击按钮就"完成" | 无法验证用户真的练了 |
| 排行榜/PK | 比拼训练次数/时长 | 数据是假的，排名无意义 |
| 成就徽章 | 基于假的训练数据 | 激励机制建立在虚假基础上 |
| 挑战赛 | 同上 | 用户可以"作弊"完成 |
| 会员吸引力 | 解锁"执行功能" | 执行流程本身缺乏价值 |

### 新产品定位

**FitPlan Pro = 智能健身知识平台**

```
旧版本：陪你练 → 点击完成 → 假成就 → 无意义排名
新版本：教你练 → 学习知识 → 自己实践 → 记录成长
```

**核心价值主张**：
- 用户来这里**学习**怎么锻炼，然后自己去健身房/家里练
- 知识库为核心，AI为辅助增强
- 不做虚假的"完成流程"，而是做真实的"知识传授"

## 二、核心功能架构

### 2.1 动作百科（核心模块）

**定位**：全站最核心的内容模块，用户学习动作的权威来源

| 功能 | 说明 |
|------|------|
| 动作列表 | 300+ 动作，按肌群/器械/难度分类 |
| 动作详情页 | 名称、目标肌群、难度、器械需求 |
| 动作要点 | 核心技巧说明（文字） |
| 常见错误 | 新手易犯错误及纠正方法 |
| 替代动作 | 同肌群的替代方案 |
| 图片/视频演示 | 动作示范（可先用静态图） |

**数据来源**：现有 `exercises.ts` + 扩展文件，300+ 动作已具备基础信息

### 2.2 训练知识库

**定位**：用户学习训练原理的系统化内容

| 类别 | 内容示例 |
|------|----------|
| **训练原理** | 增肌原理、减脂原理、力量训练原理、耐力训练 |
| **周期化训练** | 训练周期设计、渐进超负荷、休息恢复 |
| **人群指南** | 新手入门、女性训练、中年训练、老年训练 |
| **专项训练** | 核心训练、功能性训练、运动康复 |

**文章结构**：
- 标题 + 概述
- 正文内容（Markdown）
- 关键要点总结
- 相关动作推荐

### 2.3 饮食营养指南

**定位**：健身饮食的系统化知识

| 类别 | 内容示例 |
|------|----------|
| **营养基础** | 蛋白质、碳水化合物、脂肪的作用 |
| **增肌饮食** | 增肌期热量计算、蛋白质摄入量、饮食安排 |
| **减脂饮食** | 减脂期热量控制、饮食策略、常见误区 |
| **补剂指南** | 蛋白粉、肌酸、BCAA等补剂介绍 |

### 2.4 AI 智能功能

**定位**：AI作为知识平台的增强辅助，而非核心

| 功能 | 说明 | 会员权限 | 实现阶段 |
|------|------|----------|----------|
| **AI 计划生成** | 根据用户条件生成个性化训练计划模板 | Free: 1次/天，Pro: 无限 | Phase 1（已有） |
| **AI 知识问答** | 用户问健身相关问题，AI回答 | Pro: 无限 | Phase 3 |
| **AI 饮食建议** | 根据用户目标和数据生成饮食建议 | Pro only | Phase 3 |
| **AI 文章生成** | AI生成知识文章初稿，人工审核后发布 | 内部工具 | Phase 2 |

**AI API 来源**：现有 Wenwen AI (Claude Haiku)，可继续使用

### 2.5 训练计划模板

**定位**：参考性质的模板，而非"执行流程"

| 功能 | 说明 |
|------|------|
| 预设计划 | 10个预设计划作为学习模板 |
| AI生成计划 | 用户条件生成个性化模板 |
| 计划结构学习 | 用户了解计划的周数、频率、动作安排原理 |
| 导出/打印 | Pro: 导出PDF或打印计划 |

**关键改造**：
- 移除"开始训练"的虚假执行流程
- 改为"查看计划结构"的学习模式
- 用户看完后自己去实践

### 2.6 用户数据记录（可选功能）

**定位**：真实的数据记录，而非虚假的"完成"

| 功能 | 说明 |
|------|------|
| 训练日志 | 用户手动记录今天的训练内容（可选） |
| 身体数据 | 体重、体脂等数据记录 |
| 力量进步 | 最大重量、1RM记录 |
| 训练日历 | 记录可视化 |

**关键改造**：
- 移除"点击完成"的流程
- 改为"手动记录"的日志系统
- 用户自己决定是否记录，不强制

### 2.7 社区（改造）

**定位**：知识分享和问答交流

| 功能 | 说明 |
|------|------|
| 学习心得分享 | 分享学习动作/文章的心得 |
| 训练经验分享 | 分享自己的训练经验 |
| 问题提问 | 向社区提问健身相关问题 |
| 回答交流 | 其他用户回答问题 |

**关键改造**：
- 移除"打卡签到"性质的动态
- 改为"知识交流"性质的社区
- 增加问答功能

### 2.8 Pro 会员价值

**定位**：解锁高级知识内容和AI功能

| 权限 | Free | Pro ($9.99/月) |
|------|------|----------------|
| 动作百科基础内容 | ✅ 全部 | ✅ 全部 |
| 动作详细教学内容 | ❌ 部分 | ✅ 全部 |
| 训练知识库基础文章 | ✅ 部分 | ✅ 全部 |
| 饮食营养指南 | ❌ 无 | ✅ 全部 |
| AI 计划生成 | 1次/天 | 无限 |
| AI 知识问答 | ❌ 无 | ✅ 无限 |
| AI 饮食建议 | ❌ 无 | ✅ 无限 |
| 数据导出 | ❌ 无 | ✅ 支持 |

## 七、开发优先级

### Phase 1: 基础改造 ✅ 已完成

1. ✅ 删除虚假功能（排行榜、PK、成就、挑战赛）
2. ✅ 改造首页定位（知识平台展示）
3. ✅ 改造训练计划详情页（移除执行流程）
4. ✅ 创建知识库基础框架

### Phase 2: 知识内容建设 ⏳ 待做

1. 创建训练知识库模块（扩充文章）
2. 创建饮食营养指南模块
3. 填充基础文章内容（10-20篇）

### Phase 3: AI 功能增强 ⏳ 待做

1. AI 知识问答功能（用户可向AI提问健身问题）
2. AI 饮食建议功能（根据用户目标生成饮食建议）
3. AI 文章生成工具（内部使用，生成文章初稿）

### Phase 4: 社区改造 ⏳ 待做

1. 改造社区为知识分享模式
2. 增加问答功能（AI可自动回答，用户也可互助回答）
3. 移除打卡性质内容
4. AI问答集成（Pro会员可向AI提问，AI回答会显示在问答区）
